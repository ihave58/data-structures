Create an Android application using the latest SDK with the following features:
    - The application has UI and a background Service architecture.
    - The user should be able start an audio capture from the microphone. It should be able to pause and resume the capturing from the UI.
    - Previously recorded audio will be listed and can be deleted, renamed or managed in the UI.
    - This user should be able to extract audio parameters from these audio recordings.
    - This user should be able to see the extracted audio parameters in a histogram.
